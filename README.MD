# 🚀 I_WIN – Advanced Pomodoro Productivity Web App

A modern, fully-featured Pomodoro productivity web application designed to help users stay focused, manage tasks efficiently, and track progress — all in a beautiful, immersive UI.

🔗 **Live Demo:** https://iwin-promodoro.netlify.app  

---

## 📌 Overview

**I_WIN** is a powerful, offline-first productivity tool that combines the Pomodoro technique with advanced task management, analytics, and motivational features.  
Built using **pure HTML, CSS, and JavaScript**, this app runs entirely in the browser with no backend dependencies.

---

## ✨ Key Features

### ⏱️ 1. Advanced Pomodoro Timer
- 25 min Work / 5 min Short Break / 15 min Long Break
- Fully customizable timer durations
- Auto cycle with long break after every 4 sessions
- Pause with reflection note
- Skip session option
- Animated progress ring
- Ticking sound + completion chime (Web Audio API)

---

### 🎯 2. Focus Mode
- Full-screen distraction-free mode
- Click anywhere to exit
- Displays live timer and active task

---

### 📝 3. Powerful Task Management
- Create tasks with:
  - Title, description
  - Priority (High / Medium / Low)
  - Category & due date
- Subtasks with checkboxes
- Estimated vs Actual Pomodoro tracking
- Recurring tasks (Daily / Weekly)
- Task progress bar
- Notes per task
- Search & category filters
- Mark tasks as complete

---

### 👤 4. Multi-User System
- Multiple user profiles
- Switch users anytime
- Data stored per user using localStorage
- First-time welcome screen

---

### 🎨 5. Modern UI & Design
- Glassmorphism cards with blur effects
- Animated aurora gradient backgrounds
- 5 themes:
  - Aurora, Nebula, Ocean, Forest, Sunset
- Smooth motion blobs
- Dark / Light mode toggle
- Micro-interactions & smooth transitions

---

### 💡 6. Motivation & UX
- Dynamic motivational quotes
- Celebration popup with confetti on completion
- Pause reflection modal
- Browser notifications support

---

### 📊 7. Analytics & Tracking
- Daily streak counter with goals
- Weekly Pomodoro chart (Chart.js)
- Category-wise productivity breakdown
- Total focus hours tracking
- Completed tasks & longest streak stats
- Session history log
- Recent wins dashboard

---

### ⚡ 8. Extra Features
- Timer linked to tasks (auto time logging)
- Export & Import data (JSON)
- Fully responsive (mobile-first)
- Keyboard-friendly interactions

---
